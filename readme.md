### 蒙特卡洛预测五年中工作日和双休日住宅区电动车负荷图

- x 轴为时间
- y 轴为负荷

#### 单次取样方法

1. 随机抽取电动车电池容量
2. 随机抽取电动车的起始充电时间
3. 随机抽取电动车的日行驶里程(计算需要充电时长)
4. 得到该次电动车一天中的充电时间段(开始时间, 结束时间)
